# Tissenbach Website

## Introduction
This repository contains the configuration and optimization settings for the Tissenbach website. The goal is to enhance the performance, security, and overall functionality of the website.

## Repository Contents
- **wp-config.php**: WordPress configuration file with database and site settings.
- **.htaccess**: Configuration file for server settings and URL redirections.
- **Performance optimizations**: Settings and configurations for caching, minification, and other performance improvements.
- **Security enhancements**: Settings for securing the WordPress site, including recommended plugins and configurations.

## Getting Started
To get started with the Tissenbach website, follow these steps:

1. Clone this repository to your local machine:
    ```sh
    git clone https://github.com/yourusername/tissenbach-website.git
    ```

2. Upload the configuration files to your WordPress installation directory.

3. Configure your `wp-config.php` file with your database credentials and site-specific settings.

4. Install and configure the recommended plugins for performance and security improvements.

## Recommended Plugins
- **WP Rocket**: For caching and performance optimization.
- **Wordfence**: For security enhancements.
- **Yoast SEO**: For search engine optimization.
- **LiteSpeed Cache**: For advanced caching if using LiteSpeed server.
